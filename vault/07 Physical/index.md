# Section 07: Physical Hardware

This is the optional physical track. Every core exercise has a deterministic software model; actual hardware work adds electrical, toolchain, procurement, and safety risks and is never required to complete the software course.

1. [[07.01 JTAG Interface]] — drive and diagnose a simulated TAP before touching pins.
2. [[07.02 FPGA Board]] — review a board as power, clock, reset, configuration, and connectivity contracts.
3. [[07.03 Hardware Bringup]] — test one layer at a time with a rollback and evidence log.

Prefer a supported development board. Designing and assembling a custom BGA board is a separate stretch capstone requiring experienced review.
